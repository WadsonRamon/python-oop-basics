# 🐍 Python OOP Basics

Este repositório contém pequenos programas desenvolvidos para praticar **Programação Orientada a Objetos (POO)** em Python.

Os exemplos exploram conceitos como:

* Classes
* Métodos
* Instâncias
* Encapsulamento básico
* Manipulação de atributos

---

# 📂 Arquivos do projeto

### conta001.py

Simulação simples de um sistema bancário usando uma classe `Conta`.

Funcionalidades implementadas:

* depósito
* saque
* transferência entre contas
* geração de extrato

A classe possui os seguintes atributos:

* número da conta
* CPF do titular
* nome do titular
* saldo

---

### classe_Tv.py

Simulação de uma televisão usando uma classe `Televisao`.

O programa permite:

* alterar canal para cima
* alterar canal para baixo

O objetivo é demonstrar:

* criação de objetos
* manipulação de atributos
* uso de métodos dentro de uma classe

---

# ▶️ Executando os programas

No terminal:

```bash
python conta001.py
```

ou

```bash
python classe_Tv.py
```
